# Image Covert

[sharp](https://github.com/lovell/sharp)を使用した画像変換アプリ

```sh
# after clone
$ npm install

# for development
$ npm run dev

# for production
$ npm run build

# build installer for Mac
$ npm run build:mac

# build portable for Mac
$ npm run build:mac:dir

# build installer for Windows
$ npm run build:win

# build portable for Windows
$ npm run build:win:dir
```
